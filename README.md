# Paddle Geometric

![paddle_logo](assets/paddle_logo.png)

> [!IMPORTANT]
> Paddle Geometric origin from [PyTorch Geometric](https://github.com/pyg-team/pytorch_geometric) and adapt for Paddle.
>
> It was developed base version 2.6.1 of PyTorch Geometric. It is recommended to install **nightly-build(develop)** Paddle before running any code in this branch.
>
> This project is currently under development, and some features may be incomplete.

## **Build and Install**

You can install Paddle Geometric through following commands.

1. Install [PaddlePaddle](https://www.paddlepaddle.org.cn/install/quick?docurl=/documentation/docs/zh/develop/install/pip/linux-pip.html).

1. Install Paddle Geometric

```bash
git clone https://github.com/PFCCLab/paddle_geometric.git
cd paddle_geometric
pip install -v .
```
